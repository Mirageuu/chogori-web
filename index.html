What is Chogori?

Chogori Project is a platform for building low-latency (μs) in-memory distributed persistent OLTP databases.
‍
Chogori is piloting project for next generation of distributed data and storage platform, taking the challenge of cloud scale, high performance, strongly consistent transaction system, with geo-distributed datasets.

‍

Why CHOGORI

feature

1. Microsecond latency high performance

- Memory centralized computing – NUMA and SIMD optimized flexible index
- Modern persist storage
- leveraging low latency persisted memory and SSD
- Kernel bypass network
- RDMA, DPDK- 0-copy message and data passing
‍

2. Innovative highly efficient distributed transaction

- Minimized network round trips for transactions
- Support External Consistency: Serializable Snapshot Isolation(SSI) and Linearizable Consistency
- Highly efficiently support cross geo-region global transactions mingled with within data center transactions.
‍

3. Even more flexible platform architecture for future multiple cloud scale services

- Pluggable multiple subsystem for various service implementations
- Chogori platform controls performance and scalability sensitive sub-systems – network, persistence, CPU scheduling, memory allocation, etc. Also provide efficient implementation for some pluggable subsystem, e.g. NUMA and SIMD optimized data index structures, etc.
- Currently focus on OLTP type of workloads, with future expansion plan for analytical workloads.
